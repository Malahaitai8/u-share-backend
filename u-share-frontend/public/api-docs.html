API 接口文档

垃圾分类管理系统的完整API接口文档，包含用户管理、NLP服务和图像识别等核心功能。

服务概览

系统架构

本系统采用微服务架构，包含以下核心服务：

  • 用户服务 - 处理用户认证、注册和个人信息管理
  • NLP服务 - 提供自然语言处理和垃圾分类识别功能
  • 图像识别服务 - 处理图片上传和图像分类任务

基础信息

Base URL
http://localhost:8080
认证方式
Bearer Token (JWT)
数据格式
JSON

用户服务

POST /api/auth/login
用户登录接口，验证用户名和密码，返回访问令牌。
请求参数
参数名 类型 必填 说明
username string 是 用户名
password string 是 密码
请求示例
{ "username": "admin", "password": "123456" }
响应示例
{ "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...", "token_type": "bearer", "user": { "id": 1, "username": "admin", "email": "admin@example.com" } }
POST /api/auth/register
用户注册接口，创建新用户账户。
请求参数
参数名 类型 必填 说明
username string 是 用户名（3-20个字符）
email string 是 邮箱地址
password string 是 密码（6-20个字符）
请求示例
{ "username": "newuser", "email": "user@example.com", "password": "password123" }
GET /api/user/profile
获取当前用户信息，需要Bearer Token认证。
请求头
Authorization: Bearer <your_token>
响应示例
{ "id": 1, "username": "admin", "email": "admin@example.com", "created_at": "2024-01-01T00:00:00Z", "updated_at": "2024-01-01T00:00:00Z" }

NLP服务

POST /api/nlp/classify
垃圾分类识别接口，根据物品描述返回对应的垃圾类别。
请求参数
参数名 类型 必填 说明
text string 是 物品描述文本
请求示例
{ "text": "一个空的塑料瓶子" }
响应示例
{ "category": "可回收垃圾", "confidence": 0.95, "description": "塑料瓶属于可回收垃圾，可以回收再利用", "tips": "请确保瓶子清洁干燥后投入可回收垃圾桶" }
POST /api/nlp/analyze
文本分析接口，提供垃圾相关的文本分析和建议。
请求参数
参数名 类型 必填 说明
text string 是 待分析的文本内容
analysis_type string 否 分析类型：sentiment, keywords, summary

图像识别服务

POST /api/image/upload
图片上传接口，支持多种图片格式，返回图片URL和基本信息。
请求参数
参数名 类型 必填 说明
file file 是 图片文件（支持jpg, png, gif等格式）
响应示例
{ "url": "https://example.com/uploads/image_123.jpg", "filename": "image_123.jpg", "size": 1024000, "mime_type": "image/jpeg", "upload_time": "2024-01-01T00:00:00Z" }
POST /api/image/classify
图像分类接口，识别图片中的垃圾类型。
请求参数
参数名 类型 必填 说明
image_url string 是 图片URL地址
响应示例
{ "category": "厨余垃圾", "confidence": 0.92, "description": "识别为食物残渣，属于厨余垃圾", "tips": "请将厨余垃圾投入专门的厨余垃圾桶", "related_items": ["苹果核", "剩菜", "果皮"] }
